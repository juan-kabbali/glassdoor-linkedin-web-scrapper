      Société

      Editeur de logiciel qui s'est positionné autour de la donnée et de ses enjeux liés à la Big Data, API, BI, Data hub etc. Cette entreprise de 30 personnes édite une solution d'intégration unique et unifiée offrant ainsi avec ses clients une possibilité nouvelle et adaptable à tout SI et tout domaine d'activité. Avec plus de 2 millions de CA, l'entreprise connait une forte croissance suite au succès de sa solution qui séduit de nombreuses entreprises.

      Ils ont pour objectif de : moderniser leur solution principale afin de continuer à satisfaire les utilisateurs de plus en plus importants, c'est pourquoi l'entreprise recherche un Ingénieur en développement Java Confirmé pour travailler sur la refonte de l'outil dans un environnement Cloud ainsi que sur l'innovation de ses solutions.

      Souhaitant moderniser leur solution, l'entreprise recherche un Ingénieur Java confirmé qui travaillerait sur la refonte de l'architecture de leur solution dans un environnement Cloud.

      Poste

      Création de poste : Refonte de l'architecture de la solution d'intégration principale dans un environnement Cloud et sur l'innovation des différentes solutions de la société

      Vos Tâches Sont
      • Analyse du cahier des charges
      • Rédaction des spécifications techniques et fonctionnelles
      • Conception de l'architecture logicielle
      • Développement de nouvelles fonctionnalités
      • Manipulation de bases de données
      • Réalisation de la veille technologique
      Environnement technique : Java 8, Eclispe, Jenkins, Junit, Webservice SOAP/REST, JSON, Framework JS, HTML, CSS SQL server, NoSQL, Cloud

      Pourquoi rejoindre cette entreprise ?
      • Environnement technologique stimulant (Big Data & Cloud)
      • Management participatif (décisions techniques collégiales)
      • Editeur basé en plein centre de Lyon
      Profil recherché

      – Vous êtes idéalement diplômé d’un Bac +5 dans l’informatique
      – Vous avez idéalement 3-4 ans d’expérience dans l’environnement Java
      – Vous avez déjà participé à la conception d’architecture logicielle
      – Le développement sur des environnements type cluster / hadoop / Big Data est un plus
      – Vous connaissez ou êtes intéressé de travailler sur des environnements cloud (GCP, Amazon,Azure)

      Compléments
      • Lieu : Lyon centre
      • Contrat : CDI
      • Salaire 40-43K selon expérience
