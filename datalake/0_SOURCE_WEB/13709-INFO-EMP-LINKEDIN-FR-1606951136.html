      Company Description

      Si vous souhaitez apporter vos compétences dans la réalisation d’un projet important, nous avons LA mission pour vous ! Nous recherchons en effet un(e) Consultant Décisionnel (H/F)

      Votre mission est pour un client d’envergure internationale , spécialisé dans les technologies permettant la gestion du paiement dans les réseaux de transport au niveau mondial. Il conçoit des systèmes innovants pouvant se connecter aux différents transports existants et développe des solutions modernes afin de faciliter la vie de usagers. Implanté dans le monde entier, il dispose d’un centre de développement digital au cœur de Besançon.

      Job Description

      Vous aurez les missions principales suivantes :

      Développer le reporting BI pour tous les projets tant en développement qu'en maintenance

      DBA pour les développements en cours sur les nouveaux projets

      Rapports BI livrés et validés

      Correction de rapports pour projets déjà sur le terrain

      Qualifications

      De formation supérieure en informatique (Ecole d’Ingénieurs), vous justifiez d’une première expérience sur un même type de poste réussie.

      ⚙️ Les Compétences Attendues Sont Les Suivantes

      ✔️Une bonne maîtrise de Superset / Jasper, Cassandra, NIFI et PostgreSQL

      ✔️La connaissance de Spark, Ambari, Zabbix et Ansible constituerait un plus
