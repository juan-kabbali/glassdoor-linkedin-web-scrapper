      Adsearch recrute pour un client final un INGENIEUR DEVELOPPEMENT JAVA /
      PYTHON H/F.

      Poste

      Vous rejoignez une start-up de 4 ans d’ancienneté qui révolutionne le
      conseil dans le secteur de la santé.
      Cette société de conseil de nouvelle génération combine Advisory, Data et
      Tech et est également incubateur d’innovations.
      Elle accompagne ses clients principalement issus du secteur de la santé
      (Pharma & Biotech, bien-être, santé animale) dans la définition de leurs
      stratégies et innovations jusqu’à leurs problématiques opérationnelles.
      Les clients sollicitent son expertise conseil, ses capacités analytiques,
      son agilité et sa volonté d’innover autour de différents sujets : big data,
      IA, algorithmes d’optimisation, mise en place agile et rapide de solutions
      digitales…
      En pleine croissance notre client vous offre un environnement de travail
      dynamique et des perspectives d’évolution rapide.
      Au sein d’une petite équipe sur Lyon vous apporterez vos compétences de
      développeur full-stack sur les plateformes SAAS.
      En tant que référent développement vous occuperez un rôle central dans le
      développement de leurs innovations technologiques web et mobile.
      Vous coordonnerez votre travail avec d’autres développeurs externes,
      copiloterez leurs choix de stack technique, d’architecture et
      d’infrastructure.
      Vous participerez ponctuellement au développement d’algorithmes
      d’optimisation et de prédiction (incluant les réseaux de neurones) sur
      certains projets.

      Vous présenterez votre travail à vos collègues lors de points
      hebdomadaires.

      Exemples de projets sur lesquels vous pourriez intervenir dans les

      Prochains Mois

      Mise en place d’une plateforme SAAS web de cartographie (en utilisant les
      API Google Maps / Mapbox)
      Mise en place d’une application Ipad (React Native) de recommandation de
      produits et actions à proposer auprès des professionnels de santé (avec
      algorithme statistique de recommandation / optimisation en backend).

      Environnement Technique

      Backend : Java (Spring) et Python (Django Rest)
      Frontendt : Javascript, React / Redux, Backobone
      Apps : React Native
      SGBD : MySQL, PostgreSQL
      Infrastructure : AWS
      En tant que développeur confirmé et référent technique vous pourrez à terme
      devenir le CTO de l’entreprise.

      Profil recherché
      De formation Ingénieure en informatique vous justifiez d’une expérience
      réussie en développement fullstack.
      Vous maîtrisez les langages Java J2EE et/ou Python, et idéalement les
      frameworks Spring/Django.
      Vous avez de l’expérience en développement frontend (Javascript, html, css)
      et avez utilisé les frameworks React et React Native.
      Votre expérience vous a également permis de découvrir plusieurs langages,
      frameworks et technologies.
      Vous maîtrisez les standards pour mise en place des API et se connecter à
      des API (Rest, Oauth2)
      Vous avez de l’expérience dans l’utilisation et la gestion de bases de
      données SQL (MySQL ou PostgreSQL).
      Accessoirement voici d’autres compétences qui seraient appréciées :
      Travail avec les librairies Python de data science / machine learning
      Gestion des instances EC2, RDS de AWS et autres outils d’administration
      (EBS, cloud Watch…)
      Bonne maîtrise de l’Anglais
      Expérience des méthodes de gestion de projet Agile / Scrum et des outils
      associés.
      Vous êtes autonome, curieux et passionné. Vous désirez constamment
      découvrir de nouvelles technologies.
      Vous souhaitez prendre des responsabilités techniques pour devenir à terme
      le CTO de l’entreprise.
      Vous êtes passionné par l’impact du digital et des data sur les industries
      traditionnelles ?
      Vous recherchez un environnement Agile où vous pourrez avoir un réel impact
